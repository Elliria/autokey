Autokey Folder¶

class autokey.model.folder.Folder(title: str, show_in_tray_menu: bool = False, path: Optional[str] = None)[source]¶

Manages a collection of subfolders/phrases/scripts, which may be associated with an abbreviation or hotkey.

add_item(item)[source]¶

Add a new script or phrase to the folder.

calculate_input(buffer)[source]¶

Calculate how many keystrokes were used in triggering this folder (if applicable).

get_backspace_count(buffer)[source]¶

Given the input buffer, calculate how many backspaces are needed to erase the text that triggered this folder.

remove_item(item)[source]¶

Removes the given phrase or script from the folder.
